This system enables insurers to:

* Sell new insurance policies
* Configure insurance products
* Calculate premium dynamically
* Perform underwriting approvals
* Issue insurance policies
* Manage claim lifecycle
* Detect fraud risk
* Enforce compliance and audit

User Roles:
Customer
Agent/Broker
Underwriter
Senior Underwriter
Claims Officer
Fraud Analyst
Finance Officer
Compliance Officer
Admin

Policy Components:

* Coverage Amount
* Riders
* Deductibles
* Term Length
* Eligibility Rules
* Pricing Rules
* Risk Multipliers

Example Rules:
IF age > 65 THEN exclude Premium Health Rider
IF vehicle_type = Sports Car THEN apply risk_multiplier
IF smoker = true THEN increase premium by 20%
IF no_claim_years > 3 THEN apply 10% discount
